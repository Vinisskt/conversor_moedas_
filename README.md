# COMO USAR:

Para compilar o projeto usando maven no diretorio root do projeto 
o pom.xml esta configurado para baixar a dependencias do projeto.

-> mvn clean compile <-

Para execultar o projeto:

-> mvn exec:java -Dexec.mainClass=com.app.Main <-

O projeto disponibiliza 6 conversoes entre EURO, DOLAR, REAL.

Para escolher digite o numero referente a conversão desejada no menu.

Apos ira pedir o valor desejado a ser convertido.
